# Esquema de datos

- `fuente.schema.yaml`: fichas `FTE-NNN`.
- `norma.schema.yaml`: fichas `NOR-NNN`.
- `curriculum.schema.yaml`: fichas `CUR-NNN`.
- `relacion.schema.yaml`: fichas `REL-NNN`.
- `analisis.schema.yaml`: notas `AN-NNN`.
- `chunk.schema.yaml`: chunks `CHUNK-NNNNN`.
- `pregunta.schema.yaml`: preguntas `PREG-NNN`.
- `tarea.schema.yaml`: tareas `TAREA-NNN`.
