¿Qué contiene el corpus?

Estructura organizada por etapas y tipos de documentos. Cada elemento tiene un ID único para facilitar la búsqueda.

  • 01_fuentes/ (BOE, BOC, Portales)
  • 02_normativa/
    • NOR-005 Decreto 30/2023 (ESO/Bach)
    • NOR-043 Decreto 211/2022 (Primaria)
    • NOR-047 Decreto 196/2022 (Infantil)
    • NOR-042 Permisos Docentes 2026
  • 03_curriculos/
    • infantil/ (CUR-034 a 036)
    • primaria/ (CUR-024 a 033)
    • eso/ (CUR-001 a 023)
    • bachillerato/ (CUR-037 a 045 + Optativas)
  • 05_relaciones/ (REL-NNN: Derogaciones/Desarrollos)
  • 07_corpus_ia/ (Resúmenes y Chunks RAG)
