# Prompts de ejemplo

Listos para copiar y pegar en cualquier asistente con la skill instalada (o pegándola como system prompt al inicio).

## Consulta básica de norma vigente

```
¿Cuál es la norma autonómica vigente que regula el currículo de ESO en Canarias?
Cita el decreto, el BOC, los reales decretos estatales que desarrolla y aclara
si la LOMLOE deroga o modifica la LOE.
```

## Vigencia y modificación

```
¿Está vigente la Ley Orgánica 5/2002 de Cualificaciones y Formación Profesional?
Si no, indica qué norma la deroga, cuándo y con cita del precepto correspondiente.
```

## Citación de un artículo concreto

```
¿Qué establece el artículo 121 de la Ley Orgánica 2/2006 de Educación sobre el
Proyecto Educativo de Centro? Devuelve el texto literal en su redacción
consolidada actual, indica qué modificación introdujo la LOMLOE y aclara cómo
se concreta en Canarias (decreto y artículo aplicable).
```

## Currículo por materia y curso

```
Para Biología y Geología en 3.º de ESO en Canarias, lista las competencias
específicas, los criterios de evaluación principales y los bloques de saberes
básicos. Cita el anexo del Decreto 30/2023 que aplica.
```

## Comparativa estatal vs autonómica

```
Compara las modalidades de Bachillerato del Real Decreto 243/2022 con la
oferta del Decreto 30/2023 en Canarias. Indica diferencias de ordenación y
cualquier especificidad autonómica con cita del capítulo o artículo.
```

## Formación Profesional

```
¿Qué cambios introduce la Ley Orgánica 3/2022 (LOFP) en la ordenación de la
Formación Profesional respecto al marco anterior (LO 5/2002)? Cita
disposiciones derogatorias y artículos clave. Aclara qué normativa autonómica
canaria sigue siendo aplicable y cuál ha quedado superada.
```

## Centros y organización

```
En Canarias, ¿qué normativa regula los reglamentos orgánicos de centros (ROC)?
Cita el decreto autonómico vigente, el ámbito de aplicación y la ley estatal
de la que deriva.
```

## Personal docente

```
¿Qué normativa regula la función pública docente en Canarias? Distingue
entre el TREBEP (norma básica estatal) y la Ley canaria correspondiente.
Cita el artículo aplicable a interinidad/sustituciones.
```

## Convivencia y derechos del alumnado

```
¿Cuál es la normativa autonómica vigente sobre convivencia escolar en
Canarias? Cita el decreto y los principales aspectos: protocolos, órganos,
medidas correctoras.
```

## Resumen IA breve, formato corto

```
Resúmeme en 5 viñetas, con citas exactas, qué cambia para un centro de
Educación Secundaria de Canarias entre el marco curricular de 2015 y el del
Decreto 30/2023.
```

## Plantilla genérica

```
Pregunta: <TU CONSULTA>

Reglas de respuesta:
- Cita siempre norma + localización (artículo, anexo o disposición).
- Distingue literal, resumen e interpretación.
- Si no hay evidencia suficiente, dilo y remite a BOE/BOC.
- Estructura la respuesta en: Respuesta breve / Fundamento normativo /
  Aplicación práctica en Canarias / Límites o aspectos a verificar.
```
