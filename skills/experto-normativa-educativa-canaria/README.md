# Skill — Experto en normativa educativa de Canarias

Skill copiable para Claude, Claude Code, ChatGPT, Gemini, Copilot, Codex, Cursor, Windsurf y cualquier otro asistente con system prompt configurable. Convierte al asistente en un experto que **cita siempre la norma exacta y la localización** (artículo, apartado, disposición o anexo), distingue literal/resumen/interpretación y no inventa vigencias.

## Archivos

- [`SKILL.md`](SKILL.md) — Skill canónica. Pégala como instrucción de sistema.
- [`INSTALACION.md`](INSTALACION.md) — Cómo instalarla en cada plataforma (Claude.ai, Claude Code, ChatGPT, Gemini, Copilot, Codex, Cursor…).
- [`prompts.md`](prompts.md) — Diez prompts de ejemplo listos para copiar y pegar.

## Versión publicada

La página estática con copia un clic está disponible en: <https://ateeducacion.github.io/normativa_educativa_canaria/skill.html>

## Uso resumido

1. Copia [`SKILL.md`](SKILL.md).
2. Pégalo como *system prompt*, *project instructions* o equivalente en tu asistente.
3. (Opcional) Añade `llms.txt` o `llms-full.txt` del repositorio como contexto adicional.
4. Pregunta usando los [prompts de ejemplo](prompts.md).
