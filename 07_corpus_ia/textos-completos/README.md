# Textos completos rápidos

Copias locales en texto plano para lectura rápida y uso por IA.

## Reglas de uso

- No sustituyen la fuente oficial.
- Siempre debe citarse la URL oficial asociada en `06_indices/textos-oficiales.yaml`.
- Si una norma cambia o se consolida de nuevo, la copia local debe regenerarse o marcarse desactualizada.
